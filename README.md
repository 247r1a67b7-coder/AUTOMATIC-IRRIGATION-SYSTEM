# 🌱 Automatic Irrigation System

This project uses ESP32, soil moisture sensors, and IoT (Blynk) to automate irrigation.

## 📌 Project Overview

The Automatic Irrigation System is designed to optimize water usage in agriculture by using sensor-based automation. The system monitors soil moisture, temperature, and humidity in real time and controls irrigation accordingly.

## 🎯 Objectives

* To automate irrigation using sensor data
* To reduce water wastage
* To monitor soil and environmental conditions
* To minimize human effort

## ⚙️ System Components

* ESP32 Microcontroller
* Soil Moisture Sensors (Resistive & Capacitive)
* DHT11 Temperature & Humidity Sensor
* Relay Module
* Water Pump
* 16x2 LCD Display

## 🔄 Working Principle

The system continuously reads soil moisture levels using sensors.

* If moisture is **low**, the pump turns **ON**
* If moisture is **sufficient**, the pump turns **OFF**

Temperature and humidity are also monitored and displayed on the LCD.
Data is sent to the Blynk IoT platform for remote monitoring.

## 🚀 Features
